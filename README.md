# Fashion E-Commerce Website – Hijab Jutti

A modern and fully responsive fashion e-commerce website for a Hijab brand.  
This project was developed using HTML, CSS, and JavaScript as part of my Frontend Development Internship at NexaSecure Tech.

The website provides a clean and elegant shopping interface where users can explore products, view product details, and interact with modern UI components.

## ✨ Features

• Modern and elegant user interface  
• Fully responsive design (Mobile, Tablet, Desktop)  
• Multi-page website (Home, About, Products, Contact)  
• Interactive product cards  
• Add to Cart and Buy Now buttons  
• Quantity selector for products  
• Product color selection with visual feedback  
• Contact form with input validation  
• Smooth navigation between pages  

## 🛠 Technologies Used

• HTML5  
• CSS3 (Flexbox and Grid Layout)  
• JavaScript (ES6)  
• Font Awesome Icons  
• Google Fonts (Playfair Display, Inter)
